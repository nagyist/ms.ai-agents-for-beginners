> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git

> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> এটি কোর্স সম্পূর্ণ করার জন্য প্রয়োজনীয় সবকিছু দ্রুত ডাউনলোড করতে সাহায্য করবে।

## 🌱 শুরু করা

এই কোর্সে AI এজেন্ট তৈরির মুল বিষয়গুলো কভার করা হয়েছে। প্রতিটি পাঠের আলাদা একটি বিষয় রয়েছে, তাই যেখানে ইচ্ছে সেখান থেকে শুরু করুন!

এই কোর্সের জন্য বহুভাষা সমর্থন রয়েছে। [এখানে উপলব্ধ ভাষাগুলো দেখুন](#-multi-language-support)।

যদি আপনি প্রথমবার জেনারেটিভ AI মডেল দিয়ে তৈরি করছেন, তাহলে আমাদের [Generative AI For Beginners](https://aka.ms/genai-beginners) কোর্স দেখুন, যেখানে ২১টি পাঠ আছে GenAI নিয়ে তৈরি করার জন্য।

এই রেপো [⭐স্টার করুন](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) এবং [fork করুন](https://github.com/microsoft/ai-agents-for-beginners/fork) কোড চালানোর জন্য।

### অন্যান্য শিক্ষার্থীদের সাথে পরিচিত হন, আপনার প্রশ্নের উত্তর পান

যদি আটকে যান বা AI এজেন্ট সাজানোর ক্ষেত্রে কোনো প্রশ্ন থাকে, তাহলে আমাদের নির্দিষ্ট Discord চ্যানেলে যোগ দিন [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)।

### যেগুলো আপনার প্রয়োজন

এই কোর্সের প্রতিটি পাঠে কোড উদাহরণ রয়েছে, যেগুলো কোড_স্যাম্পলস ফোল্ডারে পাওয়া যাবে। আপনার নিজের কপি তৈরি করতে [এই রেপোটি fork করতে পারেন](https://github.com/microsoft/ai-agents-for-beginners/fork)। 

এই কোড উদাহরণগুলো Microsoft Agent Framework ব্যবহার করে Azure AI Foundry Agent Service V2-এর সাথে:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure অ্যাকাউন্ট প্রয়োজন

এই কোর্সে Microsoft-এর নিচের AI এজেন্ট ফ্রেমওয়ার্ক এবং পরিষেবাগুলো ব্যবহার করা হয়েছে:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


এই কোর্সের কোড চালানোর আরও তথ্যের জন্য দেখুন [Course Setup](./00-course-setup/README.md)।
