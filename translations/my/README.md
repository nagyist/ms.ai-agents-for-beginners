## 🌱 စတင်ခြင်း

ဤဘောသင်တန်းတွင် AI အေးဂျင့်များ တည်ဆောက်ခြင်း၏ အခြေခံအချက်များကို လေ့လာသည်။ သင်ခန်းစာတိုင်းတွင် မိမိစိတ်ဝင်စားသည့် ခေါင်းစဉ်ကို စတင်လေ့လာနိုင်သည်။

ဤဘောသင်တန်းအတွက် ဘာသာစကားစုံ ထောက်ပံ့မှု ရှိသည်။ ကျွန်ုပ်တို့၏ [ရရှိနိုင်သော ဘာသာစကားများ](#-multi-language-support) သို့ သွားကြည့်ပါ။

Generative AI မော်ဒယ်များနှင့် ပထမဆုံးအကြိမ် တည်ဆောက်မှုဖြစ်ပါက၊ ကျွန်ုပ်တို့၏ [စတင်လေ့လာသူများအတွက် Generative AI](https://aka.ms/genai-beginners) ဘောသင်တန်းကို ကြည့်ပါ၊ ၎င်းတွင် GenAI ဖြင့် တည်ဆောက်မှုအပေါ် သင်ခန်းစာ ၂၁ ချက် ပါဝင်သည်။

ဤရိုက်ပို့ကို [ကြယ် (🌟) ဖြင့် အမှတ်အသား ချခြင်း](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) နှင့် [ဖေါ့်ခ် လုပ်ခြင်း](https://github.com/microsoft/ai-agents-for-beginners/fork) ဖြင့် အကုဒ် လည်ပတ်နိုင်ရန် မမေ့ပါနှင့်။

### အခြားလေ့လာသူများနှင့် တွေ့ဆုံပြီး မေးခွန်းများကို ဖြေရှင်းပါ

AI အေးဂျင့်များ ဆောက်လုပ်ခြင်း ပြဿနာများ ရှိပါက သို့မဟုတ် မေးခွန်းများ ရှိပါက၊ ကျွန်ုပ်တို့၏ Microsoft Foundry Discord ရဲ့ သီးသန့် Discord ချန်နယ်တွင် ဝင်ကြည့်ပါ။ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)

### လိုအပ်ချက်များ

ဤဘောသင်တန်း၏ သင်ခန်းစာတိုင်းတွင် ကုဒ်နမူနာများ ပါဝင်ပြီး၊ ၎င်းများကို code_samples ဖိုလ်ဒါတွင် ရှာဖွေနိုင်သည်။ ကိုယ်ပိုင် မိတ္တူတစ်ခု တည်ဆောက်ရန် [ဤရိုက်ပို့ကို ဖေါ့ခ်လုပ်ပါ](https://github.com/microsoft/ai-agents-for-beginners/fork)။

ဤလေ့ကျင့်ခန်းများတွင် ကုဒ်နမူနာများသည် Microsoft Agent Framework နှင့် Azure AI Foundry Agent Service V2 ကို အသုံးပြုသည်-

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure အကောင့် လိုအပ်သည်

ဤဘောသင်တန်းတွင် Microsoft မှ AI အေးဂျင့် ဖရိမ်းဝပ်များနှင့် ဝန်ဆောင်မှုများကို အသုံးပြုသည်-

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

ဤဘောသင်တန်း၏ ကုဒ်များ လည်ပတ်မှုပိုမိုသိရှိရန်၊ [Course Setup](./00-course-setup/README.md) သို့ သွားပါ။
