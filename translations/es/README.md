>
> Este repositorio incluye más de 50 traducciones de idiomas que aumentan significativamente el tamaño de descarga. Para clonar sin traducciones, usa sparse checkout:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Esto te brinda todo lo que necesitas para completar el curso con una descarga mucho más rápida.

## 🌱 Comenzando

Este curso tiene lecciones que cubren los fundamentos para construir Agentes de IA. Cada lección cubre su propio tema, ¡así que comienza donde prefieras!

Hay soporte multilingüe para este curso. Ve a nuestros [idiomas disponibles aquí](#-multi-language-support).

Si es la primera vez que construyes con modelos de IA Generativa, revisa nuestro curso [IA Generativa para Principiantes](https://aka.ms/genai-beginners), que incluye 21 lecciones sobre cómo construir con GenAI.

No olvides [estrellar (🌟) este repositorio](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) y [bifurcar este repositorio](https://github.com/microsoft/ai-agents-for-beginners/fork) para ejecutar el código.

### Conoce a Otros Estudiantes, Obtén Respuestas a Tus Preguntas

Si te quedas atascado o tienes alguna pregunta sobre construir Agentes de IA, únete a nuestro canal dedicado en Discord en [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Qué Necesitas

Cada lección de este curso incluye ejemplos de código, que se pueden encontrar en la carpeta code_samples. Puedes [bifurcar este repositorio](https://github.com/microsoft/ai-agents-for-beginners/fork) para crear tu propia copia.

Los ejemplos de código en estos ejercicios utilizan Microsoft Agent Framework con Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Requiere cuenta de Azure

Este curso utiliza los siguientes frameworks y servicios de Agentes de IA de Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Para más información sobre cómo ejecutar el código de este curso, ve a la [Configuración del Curso](./00-course-setup/README.md).
