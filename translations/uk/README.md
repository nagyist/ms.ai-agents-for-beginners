>

## 🌱 Початок роботи

Цей курс має уроки, які охоплюють основи створення ІІ Агентів. Кожен урок присвячений своїй темі, тож починайте де вам зручно!

Для цього курсу доступна підтримка декількох мов. Перейдіть до наших [доступних мов тут](#-multi-language-support). 

Якщо ви вперше працюєте з генеративними ІІ моделями, ознайомтеся з нашим курсом [Generative AI For Beginners](https://aka.ms/genai-beginners), який містить 21 урок про створення з GenAI.

Не забудьте [поставити "зірку" (🌟) цьому репозиторію](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) і [форкнути цей репозиторій](https://github.com/microsoft/ai-agents-for-beginners/fork), щоб запускати код.

### Зустрічайте інших учнів, отримуйте відповіді на запитання

Якщо у вас виникли труднощі або запитання зі створення ІІ Агентів, приєднуйтеся до нашого спеціального Discord-каналу в [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Що вам потрібно

Кожен урок курсу включає приклади коду, які можна знайти в папці code_samples. Ви можете [форкнути цей репозиторій](https://github.com/microsoft/ai-agents-for-beginners/fork), щоб створити власну копію.  

Приклади коду у цих вправах використовують Microsoft Agent Framework з Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - потрібен обліковий запис Azure

Цей курс використовує наступні фреймворки і сервіси AI Агентів від Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Для отримання додаткової інформації про запуск коду для цього курсу перегляньте [Course Setup](./00-course-setup/README.md).
