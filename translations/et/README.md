## 🌱 Algus

Selles kursuses on õppetunnid, mis käsitlevad AI Agentide ehituse põhitõdesid. Iga õppetund katab oma teemat, nii et alusta sealt, kus soovid!

Kursusel on mitmekeelse toe tugi. Leia meie [saadavad keeled siit](#-mitmekeelne-toetus).

Kui ehitad töötamist Generatiivse AI mudelitega esimest korda, tutvu meie [Generatiivse AI algajatele](https://aka.ms/genai-beginners) kursusega, mis sisaldab 21 õppetundi GenAI-ga ehitamisest.

Ära unusta [anda sellele hoidlaile täht ★](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ja [fork’ida see hoidla](https://github.com/microsoft/ai-agents-for-beginners/fork), et koodi käivitada.

### Tutvu teiste õppuritega, saa vastused oma küsimustele

Kui jääd hätta või sul tekib AI Agentide ehitamise kohta küsimusi, liitu meie spetsiaalse Discordi kanaliga [Microsoft Foundry Discord’is](https://aka.ms/ai-agents/discord).

### Mida vajad

Iga kursuse õppetund sisaldab koodinäiteid, mis on leitavad kausta code_samples alt. Võid [fork’ida selle hoidla](https://github.com/microsoft/ai-agents-for-beginners/fork), et luua oma isiklik koopia.

Nendes harjutustes kasutatud koodinäited rakendavad Microsoft Agent Frameworki Azure AI Foundry Agent Service V2 abil:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) – nõutav Azure konto

See kursus kasutab Microsofti järgmisi AI Agentide raamistikke ja teenuseid:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Lisateavet selle kursuse koodi käivitamise kohta saad [Kursuse seadistuse](./00-course-setup/README.md) juhendist.
